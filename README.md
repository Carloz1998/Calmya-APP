# 💬 Calmya - Saúde Mental

O **Calmya** é um projeto de inovação tecnológica desenvolvido como aplicação web focado no cuidado e monitoramento da saúde mental. A plataforma funciona como um diário emocional inteligente, auxiliando usuários no acompanhamento do bem-estar diário e conectando-os a profissionais de saúde de forma segura.

> 🎓 **Projeto Acadêmico** desenvolvido para o curso de Análise e Desenvolvimento de Sistemas (ADS) no Senac DF (2026) — Brasília, DF.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído de forma nativa e otimizada, utilizando os pilares fundamentais do desenvolvimento web:
* **HTML5:** Estruturação semântica de toda a interface e seções.
* **CSS3:** Estilização customizada, uso de layouts responsivos (Flexbox e Grid) e paleta de cores focada em bem-estar.
* **JavaScript:** Manipulação do DOM para comportamento dinâmico e transições suaves de navegação.

---

## 📋 Funcionalidades Principais

### Para o Usuário:
* **📓 Diário Digital:** Registro de pensamentos, controle de humor e consulta fácil ao histórico emocional.
* **📊 Painel de Humor:** Gráficos intuitivos para acompanhar a evolução emocional e identificar padrões de sentimento.
* **🌿 Exercícios de Bem-estar:** Sugestões inteligentes de respiração guiada, meditações e técnicas de *grounding* para momentos de crise ou baixa de humor.
* **🔒 100% Privado:** Criptografia de ponta a ponta para garantir que apenas o usuário tenha acesso aos seus registros.

### Para Profissionais de Saúde:
* **🩺 Painel Clínico:** Ferramentas para psicólogos e psiquiatras realizarem teleconsultas integradas.
* **⚠️ Alertas de Risco:** Monitoramento ativo e validação profissional por meio de registros verificados (CRP/CRM).

### 🆘 Protocolo de Emergência Inteligência
Quando sinais de crise aguda são identificados, o Calmya atua automaticamente em três etapas:
1. **Respiração:** Abertura imediata de um exercício de respiração guiada.
2. **Contato:** Envio automático de alerta para uma pessoa de confiança cadastrada.
3. **Ajuda Direta:** Atalho rápido para discagem de serviços essenciais como o **CVV (188)** e o **SAMU (192)**.

---

## 📦 Como Executar o Projeto Localmente

Por ter sido desenvolvido de forma nativa, este projeto não necessita de instaladores, gerenciadores de pacotes ou compiladores. Para rodar na sua máquina:

1. Clone o repositório ou baixe os arquivos em seu computador.
2. Localize o arquivo principal do projeto (ex: `calmya-organico.html` ou `index.html`).
3. Dê um **duplo clique** sobre o arquivo para abri-lo diretamente em qualquer navegador web atual (Chrome, Edge, Firefox, Safari).
